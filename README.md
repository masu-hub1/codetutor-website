# 🌐 CodeTutor Website

An interactive HTML, CSS, and JavaScript tutorial website with live examples and mini quizzes.

## 🚀 Features

- Sidebar navigation (HTML, CSS, JS)
- Interactive examples
- Beginner-friendly explanations
- Mini quiz section
- Dark mode toggle
- Responsive design
- Built with pure HTML, CSS, and JavaScript

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📂 Project Structure

- index.html
- style.css
- script.js

## 🎯 Purpose

Built to help beginners learn web development concepts in a simple and interactive way.
